# Build a Tribute Page

A tribute page dedicated to **Dr. Norman Borlaug**, the agronomist whose work in
wheat breeding is credited with saving a billion lives from famine. Built with
semantic HTML and CSS as a practice lab from the freeCodeCamp **Responsive Web
Design** certification.

## What it does

- Presents a titled tribute with a subtitle and a main image of Dr. Borlaug,
  complete with a descriptive caption.
- Lays out a chronological timeline of his life and achievements as a list,
  from his birth in 1914 to his death in 2009.
- Closes with a quote and a footer link to his Wikipedia entry for further
  reading (opens in a new tab).

## Tech & concepts

- **HTML5** — semantic structure with `<main>`, `<figure>`/`<figcaption>`,
  `<footer>` and an external link.
- **CSS3** — responsive image (`max-width: 100%; height: auto`), Flexbox for the
  centered figure/caption block, centered typography, rounded corners and
  constrained text width with `max-width` + `margin: 0 auto` for readability.
